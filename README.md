# Json
Json utilities for PHP
# Usage
```
1. add to composer.json
  {
      "require": {
          "duyhuynhngoc/json": "^1.0"
      }
  }

2. command line
  composer require duyhuynhngoc/json
  
3. Example
  $filename = /path/to/filename.json;
  $json = \Gemuni\Json\JSONFileReader::loadFile($filename);
```
